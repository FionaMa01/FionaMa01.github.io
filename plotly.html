Take a look at the dataset

data("rest_inspec")

It is a large dataset with 397584 observations and 18 variables. I am interested in “boro”, “cuisine description”, “grade”, “score”, “dba”, and “inspection_date” so I will select only these variables, drop missing input, and randomly sample 20000 from them.

Downsample

rest_inspec = 
rest_inspec %>%

  filter(grade %in% c("A", "B", "C"), boro != "Missing") %>% 
  drop_na() %>%

Plotly Scatterplot

scatter_plot =
  rest_inspec %>%
  filter(str_detect(cuisine_description, "[Ff][Rr][Ee][Nn][Cc][Hh]")) %>% 
  mutate(
    text_label = str_c("Grade: ", grade, "\ndba:", dba) 
    ) %>% 
  plot_ly(
    x = ~inspection_date, y = ~score, color = ~boro,
    type = "scatter", mode = "markers",
    alpha = .5, text = ~text_label
    ) %>% 
  layout(title = "The distribution of scores across inspectation dates")

scatter_plot

We couldn’t observe any clear trend from this scatterplot, indicating that scores have no clear association with the inspection date.

Plotly boxplot

boxplot =
  rest_inspec %>%
  filter(cuisine_description %in% c("French", "Korean", "American", "Asian", "African", "Chinese", "Russian", "Italian")) %>%
  mutate(
    dba = fct_reorder(dba, score)
    ) %>% 
  plot_ly(y = ~score, color = ~cuisine_description, type = "box") %>% 
  layout(title = "The distribution of scores across 8 cuisines")

boxplot

We could see that most of the cuisines performed similarily, except for African, Korean, and Russian restaurants.

Plotly barchart

barchart =
  rest_inspec %>% 
  filter(cuisine_description %in% c("French", "Korean", "American", "Asian", "African", "Chinese", "Russian", "Italian")) %>%
  count(cuisine_description) %>% 
  mutate(cuisine_description = fct_reorder(cuisine_description, n)) %>% 
  plot_ly(x = ~cuisine_description, y = ~n, color = ~cuisine_description, type = "bar", colors = "viridis") %>% 
  layout(title = "Frequency of 8 cuisines")

barchart

We could see that the number of American restaurants was the most, whereas the number of African restaurant is the least. African, Russian, and Korean restaurants had low frequency, might be a explanation for what we observed in the above boxplot for the huge variations in scores.
